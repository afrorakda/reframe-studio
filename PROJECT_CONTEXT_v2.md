# REFRAME Studio Architecture

REFRAME Studio is a library of tiny browser tools that reduce thinking friction.

Each tool transforms messy thinking into clarity.

---

## Transformation Model

All tools follow this structure:

Input → Transformation → Output

Examples:

Confusion → Question  
Problem → Options  
Idea → Angles  
Hook → Draft  
Draft → Post  

---

## Thinking Loop

REFRAME tools roughly follow this thinking cycle.

Confusion  
↓  
Question  
↓  
Options  
↓  
Idea  
↓  
Angles  
↓  
Hook  
↓  
Draft  
↓  
Publish  

Users can enter the loop anywhere.

---

## Design Philosophy

Tools must feel like:

- tiny utilities
- fast
- minimal
- frictionless

Avoid building complex apps.

Build many small tools instead.

---

## Technical Architecture

All tools are:

Single HTML files  
Vanilla JavaScript  
Client-side only  
No backend  

Tools must work on:

GitHub Pages  
Mobile browsers  
Low-spec devices  

---

## Tool Expansion Strategy

Instead of complex tools:

Build many simple transformations.

Examples:

Criticism → Lesson  
Emotion → Action  
Idea → Angles  
Problem → Options  

---

## Long-term Vision

REFRAME Studio becomes a library of thinking utilities.

Like:

Calculator → math  
REFRAME Studio → thinking
