# REFRAME Studio

REFRAME Studio is a collection of ultra-simple browser tools.

These tools are designed to reduce small everyday thinking friction.

## Principles

- 1 tool = 1 problem
- HTML + CSS + Vanilla JavaScript only
- Single HTML file
- No login
- No tracking
- No personal data collected
- Runs entirely in the browser
- Mobile friendly
