# REFRAME Mutation Map v1

REFRAME Studio is a Browser-native Thinking Playground.

AI systems operate as:

Question → Answer

REFRAME operates as:

Input → Mutation → Mutation → Mutation


--------------------------------------------------

CORE MODEL

Thought does not move between nodes.

Thought drifts inside a field.

REFRAME uses a three-field thinking model:

1. Friction Field
2. Mutation Field
3. Expression Field

Tools are not answers.

Tools are mutation contact zones inside these fields.

TOOL PRINCIPLE

1 tool = 1 primary mutation


Every tool is a Mutation Contact Zone.

A tool defines:

• Primary Friction  
• Secondary Friction  
• Primary Mutation  
• Secondary Mutation  
• Primary Expression  
• Secondary Expression


--------------------------------------------------

OUTPUT RULE

Output must not finish thinking.

Output must create the next field state.

Mutation → New Field → Next Mutation


--------------------------------------------------

DRIFT ENGINE PRINCIPLE

REFRAME does not select tools by probability.

Thought drifts through fields.

When a thought state resonates with a contact zone,
a mutation occurs.


--------------------------------------------------

MODE SYSTEM

Stable  
Drift  
Chaos


Stable
Stabilizes the field and reveals clear movement.

Drift
Introduces slight directional change.

Chaos
Creates meaningful turbulence and unexpected angles.
