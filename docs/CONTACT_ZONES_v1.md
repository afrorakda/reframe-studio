# REFRAME Contact Zones v1

REFRAME Studio tools are not answers.

Each tool is a Mutation Contact Zone inside the thinking field.


A Contact Zone defines:

• Which friction it reacts to  
• Which mutation it triggers  
• Which expression form appears  


The fields used:

Friction Field  
Mutation Field  
Expression Field

CONTACT ZONE PRINCIPLE

1 tool = 1 primary mutation


A tool does not solve a problem.

A tool creates a mutation event in the thinking field.


Mutation → New Field State → Next Mutation


REFRAME Studio exists to sustain continuous thought mutation.
